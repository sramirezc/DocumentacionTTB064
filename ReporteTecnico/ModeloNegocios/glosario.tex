\label{sec:glosario}
    En este capítulo se describen algunos términos utilizados a lo largo del documento que se considera importante detallar. 
  
\begin{description}
	\BRterm{gls:elemento}{Elemento.} Se utiliza para referirse a los casos de uso, pantallas, reglas de negocio, entidades, atributos, mensajes y actores.
	\BRterm{gls:fechaInicioProy}{Fecha de inicio del proyecto.} Es la fecha de inicio el proceso de software del sistema.
	
\end{description}
