
En este capítulo se describe cómo gestionar al Personal, lo cual corresponde a registrar Persona, modificar Persona y eliminar Persona.\\


\begin{objetivos}
	\item Describir cómo registrar una Persona.
	\item Describir cómo modificar una Persona.
	\item Describir cómo eliminar una Persona.
\end{objetivos}

\section{Gestionar Personal}

Para gestionar al Personal es necesario que solicite este operación desde el menú superior, posteriormente se mostrará la pantalla \cdtFigureRef{fig:gestionarPersonal}{Gestionar Personal}. 

\IUfig[.9]{CU2/images/gestionarPersonal}{fig:gestionarPersonal}{Gestionar Personal}

\subsubsection{Acciones}

\begin{enumerate}
	\item \cdtButton{Registrar}: Mediante este botón puede solicitar el registro de una Persona.
	\item \btnEditar[Modificar Proyecto]: Mediante este botón puede solicitar la modificación de la información de una Persona.
	\item \btnEliminar[Eliminar Proyecto]: Mediante este botón puede solicitar la eliminación de una Persona.
\end{enumerate}
