
\section{Gestionar Trayectorias}

Para gestionar las Trayectorias es necesario que solicite esta operación desde la gestión de Casos de uso como se indica en la sección \ref{cap:gestionarCasosUso}, el sistema mostrará la pantalla \cdtFigureRef{fig:gestionarTrayectorias}{Gestionar Trayectorias}.

\IUfig[.9]{CU5.1.1/images/gestionarTrayectorias}{fig:gestionarTrayectorias}{Gestionar Trayectorias}

\subsubsection{Acciones}

\begin{enumerate}
	\item \cdtButton{Regresar}: Mediante este botón puede solicitar regresar a la gestión de Casos de uso.
	\item \cdtButton{Registrar}: Mediante este botón puede solicitar el registro de una Trayectoria.
	\item \btnEditar[Modificar Trayectoria]: Mediante este botón puede solicitar la modificación de una Trayectoria.
	\item \btnEliminar[Eliminar Trayectoria]: Mediante este botón puede solicitar la eliminación de una Trayectoria.	
\end{enumerate}
