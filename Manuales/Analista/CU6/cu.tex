
En este capítulo se describe cómo gestionar las Pantallas, lo cual corresponde a registrar, modificar, eliminar y
consultar Pantalla.\\


\begin{objetivos}
	\item Describir cómo registrar una Pantalla.
	\item Describir cómo modificar una Pantalla.
	\item Describir cómo eliminar una Pantalla.
	\item Describir cómo consultar una Pantalla.
\end{objetivos}

\section{Gestionar Pantallas}

Para gestionar las Pantallas es necesario que solicite esta operación desde la gestión de Módulos como se indica en la sección \ref{cap:gestionarModulos}, el sistema mostrará la pantalla \cdtFigureRef{fig:gestionarPantallas}{Gestionar Pantallas}.

\IUfig[.9]{CU6/images/gestionarPantallas}{fig:gestionarPantallas}{Gestionar Pantallas}

\subsubsection{Acciones}

\begin{enumerate}
	\item \cdtButton{Registrar}: Mediante este botón puede solicitar el registro de una Pantalla.
	\item \btnEditar[Modificar Pantalla]: Mediante este botón puede solicitar la modificación de una Pantalla.
	\item \btnEliminar[Eliminar Pantalla]: Mediante este botón puede solicitar la eliminación de una Pantalla.
	\item \btnConsulta[Consultar Pantalla]: Mediante este botón puede solicitar la consulta de una Pantalla.
\end{enumerate}
