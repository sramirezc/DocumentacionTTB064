\subsection{Eliminar Trayectoria}

En esta subsección se describe el procedimiento a seguir para eliminar una Trayectoria, se podrá eliminar una Trayectoria cuando esta no tenga elementos asociados.

\subsubsection{Procedimiento}
\begin{enumerate}
	\item Oprima el botón \btnEliminar de la pantalla \cdtFigureRef{fig:gestionarTrayectorias}{Gestionar Trayectorias}, el sistema mostrará el mensaje de confirmación.
	
	\item Oprima el botón \cdtButton{Aceptar} del mensaje de confirmación, el sistema mostrará la pantalla \cdtFigureRef{fig:gestionarTrayectorias}{Gestionar Trayectorias} con el mensaje de éxito.
\end{enumerate}
