\section{Gestionar Casos de uso}

Para gestionar los Casos de uso es necesario que solicite esta operación desde la gestión de Módulos como se indica en la sección \ref{cap:gestionarModulos}, el sistema mostrará la pantalla \cdtFigureRef{fig:gestionarCasosUso}{Gestionar Casos de uso}.

\IUfig[.9]{CU5/images/gestionarCasosUso}{fig:gestionarCasosUso}{Gestionar Casos de uso}

\subsubsection{Acciones}

\begin{enumerate}
	\item \cdtButton{Registrar}: Mediante este botón puede solicitar el registro de un Caso de uso.
	\item \btnEditar[Modificar Caso de uso]: Mediante este botón puede solicitar la modificación de un Caso de uso.
	\item \btnTray[Gestionar Trayectorias]: Mediante este botón puede solicitar la gestión de las Trayectorias del Caso de uso.
	\item \btnExt[Gestionar Puntos de extensión]: Mediante este botón puede solicitar la gestión de los Puntos de Extensión del Caso de uso.
	\item \btnEliminar[Eliminar Caso de uso]: Mediante este botón puede solicitar la eliminación de un Caso de uso.
	\item \btnConsulta[Consultar Caso de uso]: Mediante este botón puede solicitar la consulta de un Caso de uso.
	\item \btnTerminar[Terminar Caso de uso]: Mediante este botón puede solicitar terminar un Caso de uso.
	\item \btnRevisar[Revisar Caso de uso]: Mediante este botón puede solicitar la revisión de un Caso de uso.
	\item \btnLiberar[Liberar Caso de uso]: Mediante este botón puede solicitar la revisión de un Caso de uso.
	\item \btnSolicitarCorrecciones[Solicitar correcciones del Caso de uso]: Mediante este botón puede solicitar correcciones de un Caso de uso.
	\item \btnConfigurarPrueba[Configurar Prueba]: Mediante este botón puede solicitar la configuración de la prueba de un Caso de uso.
	\item \btnGenerarPrueba[Generar Prueba]: Mediante este botón puede solicitar la generación de la prueba de un Caso de uso.
	
	
	
	
	
\end{enumerate}
