\subsection{Generar Prueba}
En esta subsección se describe el procedimiento a seguir para generar la Prueba, para realizar esta operación es necesario que configure la Prueba.

\subsubsection{Procedimiento}
\begin{enumerate}
	\item Oprima el botón \btnGenerarPrueba de la pantalla \cdtFigureRef{fig:gestionarCasosUso}{Gestionar Casos de uso}, el sistema mostrará el mensaje \cdtFigureRef{fig:cargando}{Cargando} mientras se genera la prueba. 
	      Cuando la Prueba se haya gnerado completamente, el sistema mostrará la pantalla \cdtFigureRef{fig:guardar}{Guardar Prueba}.

	%--------------------------------------------------------------------------------------------------------
	      \IUfig[.4]{CU5.8/images/cargando}{fig:cargando}{Cargando}
	%--------------------------------------------------------------------------------------------------------
	
	%--------------------------------------------------------------------------------------------------------
	      \IUfig[.6]{CU5.8/images/guardar}{fig:guardar}{Guardar Prueba}
	%--------------------------------------------------------------------------------------------------------
	
	\item Guarde el zip generado.
	
	\item Consulte el zip descargado en su sistema de archivos.
\end{enumerate}
