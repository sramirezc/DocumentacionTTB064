
En este capítulo se describe cómo gestionar los Proyectos, lo cual corresponde a elegir Colaboradores, entrar a un Proyecto y generar documento.\\


\begin{objetivos}
	\item Describir cómo elegir los Colaboradores de un Proyecto.
	\item Describir cómo entrar a un Proyecto.
	\item Describir cómo generar el documento de análisis de un Proyecto en formato PDF o DOCX.
\end{objetivos}



\section{Gestionar Proyectos}

Para gestionar Proyectos inicie sesión en el sistema como se indica en la sección \ref{cap:iniciarSesion}, posteriormente se mostrará la pantalla \cdtFigureRef{fig:gestionarProyectos}{Gestionar Proyectos}. 

\IUfig[.9]{CU4/images/gestionarProyectos}{fig:gestionarProyectos}{Gestionar Proyectos}

\subsubsection{Acciones}

\begin{enumerate}
	\item \btnColaboradores[Elegir colaboradores]: Mediante este botón puede elegir los colaboradores del Proyecto si es líder.
	\item \btnEntrar[Entrar al Proyecto]: Mediante este botón puede entrar al Proyecto.
	\item \btnPdf[Descargar Documento]: Mediante este botón puede descargar el documento de análisis en formato PDF.
	\item \btnColaboradores[Elegir colaboradores]: Mediante este botón puede descargar el documento de análisis en formato DOCX.
\end{enumerate}
