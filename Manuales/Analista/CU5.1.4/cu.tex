
\section{Gestionar Puntos de extensión}

Para gestionar los Puntos de extensión es necesario que solicite esta operación desde la gestión de Casos de uso como se indica en la sección \ref{cap:gestionarCasosUso}, el sistema mostrará la pantalla \cdtFigureRef{fig:gestionarExtension}{Gestionar Puntos de extensión}.

\IUfig[.9]{CU5.1.4/images/gestionarExtension}{fig:gestionarExtension}{Gestionar Puntos de extensión}

\subsubsection{Acciones}

\begin{enumerate}
	\item \cdtButton{Regresar}: Mediante este botón puede solicitar regresar a la gestión de Casos de uso.
	\item \cdtButton{Registrar}: Mediante este botón puede solicitar el registro de un Punto de extensión.
	\item \btnEditar[Modificar Trayectoria]: Mediante este botón puede solicitar la modificación de un Punto de extensión.
	\item \btnEliminar[Eliminar Trayectoria]: Mediante este botón puede solicitar la eliminación de un Punto de extensión.
\end{enumerate}
